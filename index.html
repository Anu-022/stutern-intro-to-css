CSS also known as Cascading Style Sheets is the language that is used to style an Html document. It describes how our Html elements will be displayed in the web browser. Things like the color, border, font-size,margin, padding are defined using css.

Margin: It is the space around an element. Margins are used to move an element up or down on a page as well as left or right. Margin is completely transparent, and it does not have any background color. It clears the area around the element.

Padding:It is the space between the element and the content inside it. It determines how elements look and sit within a container. It also shows the container background around the element in it. Padding can be affected by background colors as it clears the area around the content.

Border:The CSS border properties allow you to specify the style, width, and color of an element's border.
